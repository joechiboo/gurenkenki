# 紅蓮劍姬 - 地城試煉

單一 HTML 檔的 2D 俯視角動作遊戲。劍神流劍士深入地下城，強調打擊感（hitstop 凍幀、螢幕震動、擊退、震動回饋），支援手機觸控與鍵盤操作。

> 個人同人習作，非官方、非商業。所有程式碼與圖形皆為原創（Canvas 程序繪製，無任何官方素材）。

## 遊玩

直接用瀏覽器開啟 `index.html`，無任何依賴、無需建置。

啟用 GitHub Pages 後即可用網址遊玩（Settings → Pages → Branch: main / root）。

## 操作

| 動作 | 觸控 | 鍵盤 |
|---|---|---|
| 移動 | 左搖桿 | WASD / 方向鍵 |
| 攻擊（自動索敵、三連斬） | 「斬」鍵或點畫面 | J / 空白鍵 |
| 衝刺（無敵幀） | 任意方向快速滑動 | K / Shift |
| 奧義「光之太刀」 | 「光太刀」鍵 | Q |

## 系統

- **打擊感**：命中凍幀、螢幕震動、擊退、暴擊數字、劍氣特效、裝置震動（Android）
- **成長**:擊殺取得經驗珠，升級提升攻擊／血量上限／防禦並回血；死亡不掉等級（roguelite）
- **裝備**：精英與頭目掉落。劍＝攻擊 +15%、範圍 +8%／件；甲＝減傷 7%、血上限 +12／件。死亡後歸零
- **頭目**：每 5 層「地城守衛」（衝撞＋環形彈幕）；每 10 層「深淵稜鏡」（環繞移動＋三向彈＋三連突進）
- **奧義**：光之太刀——時停、蓄力、瞬身連斬留下殘光、收劍後傷害一齊結算，冷卻 12 秒（環形倒數）
- **RWD**：畫布依裝置長寬比動態填滿（手機直／橫、平板、桌面），平板自動放大觸控元件
